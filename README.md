## Requirements

### Backend
- [Node.js](https://nodejs.org/)
- [Express](http://expressjs.com/)
- [MySQL](https://www.mysql.com/)
- [Bookshelf.js](http://bookshelfjs.org/)

### Frontend
- [AngularJS](https://angularjs.org/)

### Utilities
- [Bower](http://bower.io/)
- [npm](https://www.npmjs.com/)

### Dependencies and Running

```
npm install
bower install
```
